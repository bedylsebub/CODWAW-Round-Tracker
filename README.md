# CODWAW-Round-Tracker
A simple round tracking script for COD World at War zombies



HOW TO USE:

1. Download the bdb_launcher
2. find your mods folder for waw
3. drag and drop bdb_launcher into mods folder
4. open steam
5. find waw and it's launch options
6. +set fs_game mods/bdb_launcher


Using these launch options will ensure that every time WAW is loaded, the mod will load alongside with it without having to manually do it.
